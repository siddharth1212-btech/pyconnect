# 💬 PyConnect

A modern desktop chat application built using **Python**, **CustomTkinter**, **SQLite**, and **Socket Programming**.

# 📌 Features

- 🔐 User Login
- 📝 User Registration
- 🔒 Password Encryption (bcrypt)
- 💬 Real-Time Chat
- 👥 Online Users
- 😀 Emoji Picker
- 📷 Image Sharing
- ✍️ Typing Indicator
- 🌙 Dark Theme
- 🖥️ Modern Desktop UI
- 🗂 SQLite Database

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/siddharth1212-btech/pyconnect.git
```

Move into project

```bash
cd pyconnect
```

Install requirements

```bash
pip install -r requirements.txt
```

Run Server

```bash
python -m server.server
```

Run Application

```bash
python main.py
```
